Game Framework - 实体

发表于2023-04-28|更新于2023-04-28|C#Game Framework
|字数总计:155|阅读时长:1分钟|阅读量:

终于到了实体了, 真不容易…

我们将游戏场景中，动态创建的一切物体定义为实体。此模块提供管理实体和实体组的功能，如显示隐藏实体、挂接实体（如挂接武器、坐骑，或者抓起另一个实体）等。实体使用结束后可以不立刻销毁，从而等待下一次重新使用。

层级如下

  • EntityManager
    • EntityGroup
      • IObjectPool
        • EntityInstanceObject
    • EntityInfo
      • EntityStatus
    • ShowEntityInfo

无需手动创建和释放

基本逻辑

ShowEntity(assetName,groupName) -> EntityGroup(assetName)

问题

为何 AudioSource 没有使用对象池?

大概和 GF 中资产配置有关.
